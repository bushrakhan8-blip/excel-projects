# 📱 Social Media Post Analysis

## Overview
A data analysis project completed as part of a junior data analyst programme. The analysis explores a social media management company's dataset containing user posts, interactions, and engagement metrics. All tasks were performed in **Microsoft Excel** using Power Query, formulas, and data visualisation tools.

## Dataset
Two datasets were imported and analysed:
- `Mission 2 Data.txt` — post-level data including post IDs, timestamps, likes, comments, shares, and content details
- `M2 - Ratings Data.csv` — content quality ratings and influence scores per post

## Tasks Completed
- **Data Import & Cleaning** — imported data via Power Query, handled missing values, corrected data types, removed unnecessary columns, and added calculated fields
- **User Activity Analysis** — identified the top seven most active users by post/reply count
- **Time-Based Analysis** — determined the busiest day and hour for posts, and monthly trends for high-engagement posts
- **Duplicate Detection** — identified posts with duplicate Post IDs
- **Engagement Metrics** — calculated interaction scores, weighted interaction scores using `SUMPRODUCT`, and post urgency categories based on time of day
- **Conditional Formatting** — applied colour-coded formatting to content quality ratings
- **Advanced Formulas** — used `VLOOKUP`, `SUMIF`, `AVERAGEIF`, `COUNTIF`, and `SUMPRODUCT` to derive insights across sheets
- **Data Visualisation** — created a line graph tracking high-engagement posts across months in 2016

## Tools & Skills
`Microsoft Excel` · `Power Query` · `VLOOKUP` · `SUMIF / AVERAGEIF / COUNTIF` · `SUMPRODUCT` · `Conditional Formatting` · `Data Cleaning` · `Data Visualisation`
